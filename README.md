# 📊 亚马逊运营能力测试系统

一个专业的亚马逊运营能力在线测评工具，包含30道精选题目，根据运营经验自适应难度，带详细答案解析。

## ✨ 功能特点

- 🎯 **智能选题**：根据运营经验自动匹配不同难度题目
- 📈 **难度权重**：5个难度等级，越难的题目分值越高
- 🏆 **评级系统**：S/A/B/C/D 五级能力评估
- 📝 **答案解析**：每题都有详细解析，边测边学
- ❌ **错题回顾**：专门查看做错的题目，查漏补缺
- 📱 **响应式设计**：支持电脑和手机访问
